# 문제

초기에 $n+1$개의 집합이 존재한다.

- `{0}, {1}, ... , {n}`

다음 쿼리를 처리하는 프로그램을 작성해야 한다.
- 1. 합집합 연산
- 2. 두 원소의 집합이 같은지 확인

## 구현해야하는 함수

다음과 같은 기본 코드가 제공된다.
```luau
local module = {}

function module.solve(n : number, queries)

end

return module
```

## 함수 설명

### 매개 변수
- `n` : 원소의 개수
- `queries` : 연산의 목록

각 연산은 다음과 같은 형식이다.
- `{0, a, b}` : `a`가 속한 집합과 `b`가 속한 집합을 합친다.
- `{1, a, b}` : `a`와 `b`가 같은 집합인지 확인한다.

### 반환값

함수는 다음을 반환해야 한다.
`{"YES","NO", ... `

즉 type 1 연산의 결과를 순서대로 저장한 배열을 반환한다.
- 같은 집합이면 "YES", 아니면 "NO"를 저장해야한다.

## 제한

- $ 1 \le n \le 1000000 $
- $ 1 \le |queries| \le 100000$
- $ 0 \le a, b \le n $

## 서브 태스크

- 서브 태스크 1 (9점) : $n <= 1000$
- 서브 태스크 2 (11점) : $n <= 10000$
- 서브 태스크 4 (80점) : 전체 제약
