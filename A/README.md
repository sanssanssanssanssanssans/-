# 문제

당신은 새로운 문자열 압축 알고리즘을 개발했다. 이 알고리즘은 연속해서 반복되는 동일한 대문자 알파벳을 하나의 문자와 반복 횟수로 표현한다.

예시를 한 번 보자.
- AAABBBBB -> A3B5
- CCCC -> C4

압축된 문자열이 주어지면, 그 문자열을 복원하는 함수 `decompress(s : string)`을 구현하자.

# 함수 설명
- 매개변수 `s`는 압축된 문자열이며, 항상 대문자 알파벳 A-Z로 시작한다.
- 각 알파벳 뒤에는 1이상, $10^9$ 이하의 정수가 붙는다.

- 압축을 해제한 원래 문자열을 반환해야한다.

# 서브태스크
- Subtask 1 (8점) : 각 반복 횟수는 9보다 작거나 같고, 서로 다른 문자가 연속해서 등장하지 않는다.
- Subtask 2 (12점) : 각 반복 횟수는 1000보다 작거나 같고, $|s| \le 1000$ 이다.
- Subtask 3 (37점) : 각 반복 횟수는 $10^6$보다 작거나 같고, 복원된 문자열의 길이는 $10^6$보다 작거나 같다.
- Subtask 4 (43점) : 각 반복 횟수는 $10^9$보다 작거나 같고, $|s| \le 1000$, 복원된 문자열의 길이는 $10^7$보다 작거나 같다.
