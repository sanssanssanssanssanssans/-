# 문제

길이 $N$의 정수 배열 $A$가 있따. 압축 알고리즘 `XORRRR`은 다음과 같이 배열을 압축한다.
- $B_1 = A_1$
- $B_i = A_i \oplus A_{i-1} (2 \le i \le N)$

여기서 $\oplus$ 연산은 비트 단위 xor 연산을 의미한다.

압축된 배열 $B$가 주어질 때, $A$를 복원하는 함수를 구현하자.

## 구현해야하는 함수

다음과 같은 기본 코드가 제공된다.
```luau
local module = {}

function module.decode(b : {number})

end

return module
```

## 함수 설명

- 매개변수 $b$는 압축된 배열 $B$이다.
- $1 \le N \le 2 \times 10^5$
- $0 \le B_i < 2^{31}$
- 원래 배열 $A$를 반환 하면 된다.

## 예제
`[5, 2, 0, 5]` -> `[5, 7, 7, 2]`

## 서브 태스크

- 서브 태스크 1 (10점) : $N = 1$
- 서브 태스크 2 (20점) : $N = 1000$
- 서브 태스크 3 (30점) : $0 \le B_i < 2^{16}$
- 서브 태스크 4 (40점) : 전체 제약
